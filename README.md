# Pneumologia PE - Carpina

Landing page da clínica Carpina preparada para GitHub Pages.

Domínio previsto: https://carpina.pneumologia-pe.com.br

## Estrutura

- index.html: landing da clínica.
- CNAME: domínio customizado do GitHub Pages.
- pre-atendimento/index.html: rota pública alinhada ao domínio, redirecionando para a ficha central com clinica=carpina.
- preatendimento/index.html: compatibilidade com links antigos sem hífen.

## Publicação

GitHub Pages deve publicar a branch main a partir da raiz /.
